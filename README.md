# My Dotfiles

## Requirements 

Ensure you have the following installed on your system

### Git 
```
sudo pacman -S git 
``` 

### Github Cli (optional)

``` 
sudo pacman -S github-cli
```

### Stow 

``` 
sudo pacman -S stow 
```

## Installation 
```
$ gh repo clone Buppy-212/Dotfiles (github-cli way)
$ https://github.com/Buppy-212/Dotfiles.git (normal way)
$ cd dotfiles 
$ stow .
```
