## Installation

Ensure you have the following installed on your system

```Bash
sudo pacman -S git stow
```

Then run the following commands from the home directory

```Bash
git clone https://github.com/Buppy-212/Dotfiles.git
cd Dotfiles
stow --dotfiles .
```
